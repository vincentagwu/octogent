# Contributing

## Prerequisites

- Node.js `22+`
- pnpm

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Testing

```bash
pnpm test
```

## Linting and formatting

```bash
pnpm lint
pnpm format
```

## Pull request expectations

- Keep changes incremental and test-backed.
- Update docs for behavior or workflow changes.
- Keep `context/` notes current when architecture decisions shift.
